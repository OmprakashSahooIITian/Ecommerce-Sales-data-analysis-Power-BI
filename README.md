E-Commerce Sales Data Analysis

  Project Overview

This project focuses on analyzing e-commerce sales data to understand business performance, identify revenue trends, and uncover actionable insights. The analysis transforms raw transactional data into an interactive dashboard that supports data-driven decision-making for sales, product, and regional strategy.

  Objectives

Analyze overall sales and profit performance

Identify top-selling products and high-revenue categories

Track sales trends over time

Evaluate regional and customer segment performance

Provide insights to support business optimization

  Dataset Description

The dataset contains order-level e-commerce transactions, including:

Order details (Order ID, Order Date)

Customer and segment information

Product, category, and sub-category data

Sales, quantity, discount, and profit metrics

  Data Cleaning & Preparation

Data was cleaned and transformed using Power BI Power Query:

Removed duplicate records

Standardized date formats and text fields

Handled missing and invalid values

Corrected data types

Created derived date features for trend analysis

  Key Metrics & Analysis

Total Sales

Total Profit

Profit Margin

Order Volume

Category, product, and regional performance

  Dashboard Features

KPI cards for high-level performance

Time-series sales trend analysis

Category and product-level comparisons

Regional sales distribution

Interactive slicers for dynamic filtering

  Tools Used

Power BI Desktop

Power Query

DAX

  Key Insights

The dashboard highlights major revenue drivers, regional performance differences, sales growth patterns, and product-level contributions, enabling stakeholders to make informed strategic decisions.
